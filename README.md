# 简介
视频解码，使用wasm，调用ffmpeg库。  

# 依赖
* [Emscripten](https://emscripten.org/docs/getting_started/downloads.html)  
* [FFmpeg](https://ffmpeg.org/download.html#get-sources)  

# 构建
./build.sh

# 使用
见 [示例项目](https://github.com/zhaohuijun/video-decoder-test)
